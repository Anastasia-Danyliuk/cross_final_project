# Before checking app, https://cors-anywhere.herokuapp.com/corsdemo click button /Request temporary access to demo server/

### Вдосконалено дизайн та кольорову схему застосунку з повною підтримкою світлої та темної теми для зниження напруги очей користувачів, які використовують застосунок у вечірній або денний час.
### Ключове розширення: інтеграція нової вкладки Погоди, яка тепер рекомендує музичні треки, ґрунтуючись на погоді. Це додає персоналізації застосунку. Наприклад, у похмуру погоду пропонуються "меланхолійні" або "затишні" треки. Це створює унікальність застосунку.
### Для покращення користувацького досвіду додано анімаційний фідбек при взаємодії з елементами. Це дає користувачеві миттєвий візуальний фідбек, підтверджуючи, що його дія (лайк/збереження) була успішно зареєстрована.

###### Цей мобільний музичний застосунок створений на React Native з використанням Redux та Context API. Основна функціональність: прослуховування, збереження та вподобання музичних треків з можливістю видалення та перегляду у власній бібліотеці. Для зручної роботи з даними використано Redux Toolkit: списки збережених (Saved) та вподобаних (Liked) треків доступні глобально і оновлюються без перезапуску екранів.
###### Додано повну підтримку світлої та темної теми, перемикання якої реалізоване через Theme Context. Було вдосконалено дизайн, додано анімаційний фідбек при взаємодії з елементами (лайк, збереження, видалення), що підвищує зручність користування. Також інтегровано вкладку з погодою, яка пропонує музичні рекомендації відповідно до погодних умов та додає персоналізації.
